# Kairos.extension

> Plugin Estagiário para o Revit — Ecossistema Kairós
>
> ## Sobre o projeto
>
> O `Kairos.extension` é um plugin para o Autodesk Revit desenvolvido como parte do ecossistema Kairós. Utiliza o framework **pyRevit** para estender as funcionalidades do Revit com ferramentas personalizadas para escritórios de arquitetura.
>
> ## Tecnologias
>
> - **Linguagem:** Python
> - - **Framework:** pyRevit
>   - - **Plataforma:** Autodesk Revit
>    
>     - ## Estrutura do projeto
>    
>     - ```
>       Kairos.extension/
>       ├── Estagiário.tab/   # Aba principal do plugin no Revit
>       ├── lib/              # Bibliotecas e módulos auxiliares
>       ├── hooks/            # Hooks de eventos do Revit
>       └── LICENSE
>       ```
>
> ## Como instalar
>
> 1. Instale o [pyRevit](https://github.com/eirannejad/pyRevit/releases)
> 2. 2. Copie a pasta `Kairos.extension` para o diretório de extensões do pyRevit
>    3. 3. Recarregue o pyRevit dentro do Revit
